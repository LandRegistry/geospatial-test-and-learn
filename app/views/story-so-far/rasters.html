Rasters

The aim of this piece of work is to "develop and prove a method for transforming scanned spatial data into machine interpretable spatial data (digitising raster titles)"

What is the problem?

More than half of our spatial data isn't currently held in a digital format. Over 50% of HMLRs spatial data still exists in a format which is not machine interpretable, not machine readable and not able to be used with modern digital tools. This is split primarily between things which are stored on paper and things which are stored as scanned documents - esstentially as useful as paper, with the exception that you can view it anywhere at any time.

This is quite a big deal for the ambition of having a geospatial register as it means over half of our spatial data cannot currently be used. To fix this, that data needs to be digitized and made to be machine readable.
