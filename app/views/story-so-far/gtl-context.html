DRAFT CONTENT

The aim of the test and learn work is to build the capabilities required for a Geospatial Register. This will be acheived by transforming the data in a 5K sample area of Bristol.

There are 6 key areas of the work which support this aim

  • Create data from raster title plans
  • Convert vector (TMF) data to an open data format
  • Match textual and spatial register data
  • Extract text from deeds
  • Join text from deeds to the combined textual and spatial data
  • Visualise the result
